# 安全境界（Safety Boundary）

> このロボットは医療機器ではありません。看護師が安全に介助へ入るための「準備支援」に特化します。

---

## ロボットがやること

| アクション | 詳細 |
|-----------|------|
| ✅ 患者リクエストの受信 | ボタン押下を検知してタスク開始 |
| ✅ キットの選択と配送 | リクエスト種別に応じてキットをベッドサイドへ届ける |
| ✅ QR/ArUcoで患者ID照合 | 正しい患者に正しいキットを届ける |
| ✅ トレイの高さ調整 | 看護師が使いやすい高さでトレイを提示 |
| ✅ 患者への待機メッセージ | 「立ち上がらずお待ちください」を表示 |
| ✅ 看護師確認後にキット開放 | 確認ボタンが押されるまでロック |
| ✅ 動作ログの記録 | 時刻・患者ID・状態・結果を保存 |

## ロボットがやらないこと

| 禁止アクション | 理由 |
|--------------|------|
| ❌ 患者の身体を支える | 転倒・骨折リスク、医療資格が必要 |
| ❌ 患者を立たせる・移乗介助 | 同上 |
| ❌ 尿器・便器を患者に当てる | 身体への直接接触 |
| ❌ 点滴ルート・薬剤に触れる | 薬剤事故・感染リスク |
| ❌ 医療判断をする | 看護師の領域 |
| ❌ 看護師確認なしにキットを開放 | 誤配送・安全確認なし介助の防止 |

## 緊急停止条件

以下の場合は即座に停止し、看護師ダッシュボードにアラートを送信する：

- 障害物センサーが反応した場合
- 患者IDの照合に失敗した場合
- 看護師からの停止指示を受信した場合
- モーター異常・通信断などのエラーを検知した場合

**緊急停止後は必ず人間の確認を経てから再起動する。**

## 責任の範囲

| 行為 | 責任者 |
|------|--------|
| ロボットの移動・停止・トレイ操作 | ロボットシステム（チーム責任） |
| 患者の安全確認・実際の介助 | 看護師 |
| キット内容の確認・補充 | 看護師・病棟スタッフ |
| ロボット異常後の対応 | 看護師・チーム |
